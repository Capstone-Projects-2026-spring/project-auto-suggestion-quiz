# Features and Requirements


## Features
- Role-based accounts (student, teacher, admin)
- Teacher-created coding quizzes with test cases and starter code
- AI next-line code suggestions with multiple interaction modes
- Integrated code editor and terminal
- Student progress tracking and analytics dashboard
- Anti-paste and integrity controls
- Multi-language support (Python, Java, C, JavaScript)

## Functional Requirements
- Secure authentication and Edu ID login
- Quiz creation and configuration by instructors
- Student evaluation of code suggestions
- Logging of student decisions and progress
- Resume functionality for unfinished quizzes

## Nonfunctional Requirements
- Real-time responsiveness
- Secure data storage and role-based access control
- Cloud-based scalability
- Cross-device accessibility
- Intuitive and learning-focused UI

## Tech Stack (Example)
- Frontend: React / Monaco Editor
- Backend: Node.js / Express
- Database: PostgreSQL / MongoDB
- AI Suggestions: API-based model
- Deployment: Cloud platform (AWS, Azure, or GCP)
