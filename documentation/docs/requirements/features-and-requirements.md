## Functional Requirements

- Role-based accounts (student, teacher, admin)
- Secure authentication with Edu ID login
- Integrated code editor
- Code execution terminal (sandboxed environment)
- AI code suggestion system (next-line suggestions)
- Teacher-created coding quizzes with test cases and starter code
- Resume functionality for unfinished quizzes
- Logging of student actions and decisions
- Multiple AI interaction modes (hint, explanation, debugging, autocomplete)
- Student evaluation of AI suggestions
- Student progress tracking
- Teacher analytics dashboard
- Quiz configuration options (time limits, attempts, visibility)
- Multi-language support (Python, Java, C, JavaScript)
- Academic integrity controls (anti-paste, plagiarism detection, behavior monitoring)
- Admin tools for review and moderation

---

## Nonfunctional Requirements

- Real-time responsiveness
- Secure data storage (encryption in transit and at rest)
- Role-based access control
- Secure sandboxed code execution
- High availability and reliability
- Cloud-based scalability
- Modular and extensible system architecture
- API-based integration capability
- Cross-device accessibility
- Intuitive and learning-focused UI
- Accessibility compliance
- System logging and monitoring
- Maintainable and well-documented codebase
- Automated testing and CI/CD support
