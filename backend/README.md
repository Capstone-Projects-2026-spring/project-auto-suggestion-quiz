# Backend - AutoSuggestion Quiz API

FastAPI + SQLite backend for the AutoSuggestion Quiz application.

## Setup

```bash
cd backend
python3 -m venv venv
source venv/bin/activate  # macOS/Linux
# venv\Scripts\activate   # Windows

pip install -r requirements.txt
```

## Database

SQLite database is initialized automatically on first run via `database.py`.
